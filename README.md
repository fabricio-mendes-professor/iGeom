# iGeom
Esta é a reconstrução dos códigos do iGeomJS com o objetivo de aprendizado.
O iGeomJS é um ambiente de geometria interativa e dinâmica que roda diretamente no navegador, sem a necessidade de 
instalar programas adicionais. Ele é a versão moderna e adaptada para a web (utilizando a linguagem JavaScript) do 
clássico iGeom, um software educacional de código aberto desenvolvido pelo projeto iMática do IME-USP 
(Instituto de Matemática e Estatística da Universidade de São Paulo).
